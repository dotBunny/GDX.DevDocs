    Features

    Note

    Below is an overview of some of the notable features contained within the GDX runtime package. For a more thorough list of functionality, explore the API documentation directly.

    Extending APIs

    At the root of the GDX namespace are static extension and utility classes that build the foundation for the framework, providing missing functionality to numerous types inside the Unity API.

    C Sharp

    Extension Description
    Array A bunch of Array operations which are commonly used by other methods. For example getting the last or first index of an item or value in an Array.
    Byte Contains a simple non-allocating extension method to get a stable hash code from an array of bytes as well as an optimized comparison method.
    Enum Useful functionality around Flags
    IList A common set of methods for IList implementing collections allowing for optimized value and item checks, as well as some extras.
    String A vast collection of extension methods covering a wide range from encryption to hash codes, to simply finding content an existing string.

    GDX

    Extension Description
    Array2D A small amount of functionality around manipulating data contained within an Array2D.
    SimpleList A slightly less performant set of functionality for a SimpleList.

    Unity

    Extension Description
    Addressables A collection of useful spawning and management functionality built on top of the com.unity.addressables package. For example identifying if a AssetReference is valid and able to be instantiated.
    BoxCollider Find out if a position is contained within a BoxCollider.
    Camera Methods to force a specific Camera to render to specific outputs.
    CapsuleCollider Simple helper for finding out which way a CapsuleCollider is oriented.
    GameObject Numerous methods meant to replace commonly implemented functionality around the component system and scene operations.
    Mesh A helper to calculate the approximate volume of a mesh.
    MonoBehaviour A small set of wrapped functionality for component based operations.
    NavMeshPath Some helpful functions when trying to build out AI logic.
    Rigidbody A method for determining the moment of inertia.
    Transform Some useful functionality when it comes to working with Transform.
    Vector2 Additional functionality for common calculations done with Vector2.
    Vector3 Additional functionality for common calculations done with Vector3.

    Optimized Collections

    Located in the GDX.Collections namespace is an extensive group of structs and classes designed for performance-sensitive environments.

    Warning

    It is important to note that many of the structures backing data, indices, counts, etc., are publicly accessible. This is meant for advanced usage; change at runtime at your own risk.

    Type Description Base
    Bit Arrays A selection of sized 8, 16, 32, 64, 128, 256 and 512 index accessed bit arrays. struct
    Free List An array where indices are allocated from and stored in an in-place linked list. struct
    Native Sparse Set A Sparse Set, backed by two NativeArray. struct
    Sparse Set An adapter collection for external data arrays that allows constant-time insertion, deletion, and lookup by handle, as well as array-like iteration. struct

    Generic

    Type Description Base
    2D Array A 2-dimension array backed by a singular array. struct
    Circular Buffer A sized buffer which loops back over itself as elements are filled. class
    Native 2D Array A 2-dimension NativeArray with a xy based accessor. struct
    Native Uniform 3D Array A 3-dimension NativeArray where all axis are the same length. Numerous accessors available, including a xyz based. struct
    Serializable Dictionary A dictionary that can be serialized by Unity into its native YAML format. Usage of the editor side property drawer requires Unity 2020.1+. class
    Simple List An optimized List replacement. struct
    String Key Dictionary An optimzed Dictionary<string, T> replacement. struct

    Pooling

    Type Description Base
    Array Pool An object pool for arrays with power-of-two lengths. struct
    SimpleList Managed Pool A basic implementation of an IManagedPool backed by a SimpleList, usable by the ManagedPoolBuilder.

    Helpful Utilities

    Everyone loves static utility classes, and naturally GDX has a bunch of them to help fill out some commonly used functionality spread out across its namespace.

    Utility Description
    Display Simplified querying of display capabilities.
    Localization ISO language based formatting functionality.
    Memory Specific functionality to ensure the heap is kept as small as possible.
    Platform A bunch of Hardware and I/O related methods.
    Profiling A wrapper for Unity's builtin profiler with file management.
    Range Simplified logic for picking values within a range.
    Reflection Collection of methods to help with language level reflection.
    Rotate A set of functionality to extend on Unity's rotation based methods.
    Smooth Some fancy smoothing functions.

    Developer Toolbox

    The missing or upgraded development process functionality.

    Feature Description
    Build Info Writer A system which latches on to the build pipeline to write a configurable BuildInfo class containing a passed set of parameters from commandline arguments.
    Build Verification Report A framework for build verification tests without the need for build-specific instrumentation and inclusion of additional libraries.
    Command Line Parser A configurable command line parameter parsing system which can be queried at runtime (and in special cases during editor automation) for flags and key-value-pair data.
    Managed Resource Audit A reporting mechanism for outputting managed objects at runtime as well as differences between captures.
    NUnit Report An abstraction of the NUnit report format allowing for generating of arbitrary reports that can be captured by different systems.
    Text Generator A text file generator allowing for easy formatting and unwinding.

    Miscellaneous

    A few other notable inclusions which have been found useful.

    Feature Description
    Coalesce Stream An infinitely sized Stream.
    Tarball Support A native method to inflate .tar and .tar.gz files. Used internally in the GDX package to handle upgrades from GitHub.
    WELL1024a A deterministic random number generator.
