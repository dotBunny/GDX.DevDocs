    Field Value

    Remarks

    This is useful when using the GetInstance(Int32, Int32, Int32) and RemoveInstance(Int32) calls to maximize reuse and limiting allocations.

    A common pattern is to use the the UnityEngine.GameObject's InstanceID or an Entity Number to create a unique indexer.
