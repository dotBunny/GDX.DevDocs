    ExportToCommaSeperatedValues(DataTableBase, String)

    Export the data found in the DataTableBase to a CSV file.

    Declaration
    public static void ExportToCommaSeperatedValues(this DataTableBase dataTable, string filePath)
    Parameters
    Type Name Description
    DataTableBase dataTable

    The target DataTableBase to export data from.
    System.String filePath

    The absolute path where to write the data in CSV format.
    Remarks

    The data is able to be imported via UpdateFromCommaSeperatedValues(DataTableBase, String, Boolean), however there is a requirement that the structure of the table does not change, nor does the column order. The Row Identifier will be used to match up rows, with an option to create new rows.
