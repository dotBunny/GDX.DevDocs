    Key

    The associated System.Int32 key with the DebugLineBuffer.

    Declaration
    public readonly int Key
    Field Value
    Type Description
    System.Int32
    Remarks

    This is useful for identifying the DebugLineBuffer in different contexts; its specific use is meant for being able to recall a DebugLineBuffer from the DebugDraw.

    A common pattern is to use the the UnityEngine.GameObject's InstanceID or an Entity Number to create a unique indexer. Collisions can occur if you are not careful about how you index your DebugLineBuffer.

    Finalized

    Has the DebugLineBuffer been converged?

    Declaration
    public bool Finalized { get; }
    Property Value
    Type Description
    System.Boolean
    Remarks

    A finalized DebugLineBuffer has had its command buffer filled with the fixed draw calls based on the meshes/materials outlined. If a meshReference is invalidated by Invalidate(Int32), the DebugLineBuffer will become not finalized and will re-converge itself next Execute().
